# congenial-enigma
This repo contains scripts for pre-processing graphs including re-ordering, cleaning, ... etc

## pre-process-big-files.sh 

input: an edge list (input.txt)
output: three files 
		- input.txt.sorted.uniq: a list of uniq node ids
